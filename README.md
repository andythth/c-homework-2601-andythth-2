# 💻 [2026-01] 나만의 C언어 소프트웨어 개발 프로젝트

## 1. 시나리오 제목
* 나의 주식 현황과 수익률을 나타내고 평가해주는 시스템

<br>


## 2. 시나리오 (5~10줄)
* (이곳에 한 학기 동안 개발할 프로그램의 전체 시나리오를 개괄적으로 요약합니다.)
* 이 프로그램은 주식투자를 하는 개미를 위한 프로그램으로, 매수가격, 매수종목 보유수량, 현재가 등을 입력하면 수익률과 얼마만큼의 수익을 보았는지를 알려주는 프로그램 입니다
* 사용자는 자신의 주식 수익률을 정리하여 알수 있고 각 종목마다 수익률과 손익여부를 쉽게 확인할수 있습니다.
* 복합적인 조건(수익률, 수익(원))이 일정 값을 달성하면 각 값마다 '좋은 익절 타이밍'이나 '손해보는중'이라는 수식어를 붙여줍니다
* 최종적으로는 사용자가 각 투자 종목을 1개이상 입력해 수익률과 손익정도를 볼수 있으며, 수식어를 붙여 보다 쉽게 수익인지, 본전인지, 손해인지를 쉽게 알게 하고 싶은 마음이 들어 이번학기에 배우는 여러 명령코드를 사용해 시스템을 완성하고 싶습니다. 

<br>


## 3. 예상 기능 및 메뉴 (최소 5개)
1. 종목시장, 매수가격, 보유수량, 현재가 입력
2. 수익률, 손익정도 표시
3. 각 종목당 수익과 수익률을 계산하여 등급 판별
4. 수식어를 붙여 종목의 손익 여부를 재밌게 표현
5. 각 종목별 수익을 나타내는 결과화면에 1개이상 한번에 출력
<br>

# 🚀 [버전별 개발 일지 & AI 협업 기록]

## 🟦 [1차 과제: V1.0] 기초 뼈대 구축 (입출력과 변수)
### **✨V1.0 개발 내용:**
  * 사용자가 매수한 종목의 주식시장 알파벳 첫글자를 대문자로 입력(나중에 종목별 구체화 예정)
  * 각 매수단가, 보유수, 현재가를 입력받는 화면을 구현했습니다.
  * 사칙연산을 통해 결과화면을 구현했습니다.
  * 결과화면에는 시장종류, 매수총액, 현재가치, 수익률 계산하여 출력합니다.
    
### **🤖 AI 파트너십 과정**
 1. **내용 1**
    * **프롬프트 요약:** "C언어 4단계 과제 로드맵(입출력 ➔ 조건문 ➔ 반복문 ➔ 배열/함수)에 적합한 RPG 캐릭터 성장 시나리오 및 단계별 기능 확장 로드맵 기획 요청"
    * **적용 내용:** 단순히 수익을 계산하는구조가 아닌, 1단계(기초 투자 정보 입력) : 단일 주식 종목에 대한 기본 정보를 입력받고, 총 투자 금액과 현재가치를 계산(변수와 입출력) / 2단계( 수익률 판정 및 손절/익절 알림) : 현재 수익률에 따라 계좌의 상태를 재미있게 평가해주고, 목표 수익률 도달 여부를 판단(조건문) / 3단계(HTS/MTS 시스템 도입) : 실제 주식 앱처럼 원하는 메뉴를 선택하고 반복해서 작업을 수행할수 있는 메인 로직을 만든다(반복문과 메뉴) / 4단계(분산투자 및 코드 모듈화) : 한 종목만 거래하던것을 여러 종목으로 확장하고, 복잡해진 코드를 함수로 깔끔하게 정리한다(함수와 배열)
      
 2. **내용 2**
    * **프롬프트 요약:** "1차 과제 필수 요건(변수 5개, 자료형 3개 이상)을 충족하기 위한 캐릭터 초기 스탯 데이터 구조 모델링 논의"
    * **적용 내용:** 단순한 이득이 얼마인지 계산하는것이 아닌 변수를 사용해 int. char, float 등을 사용하여 정확히 구분하고 결과또한 값을 각각의 값을 통해 데이터의 구조적 완성도를 높임
    
### **🛠️ Troubleshooting & 기술 회고:**
  1. **문제 1:** '%c'앞의 공간에 띄어쓰기를 하지 않아 출력 화면에 엔터가 적용됨
     * **해결:** '%c' 앞에 공백을 두어 버퍼를 비우는 팁을 얻고 코드를 수정하여 해결함.
  2. **문제 2:** 계산결과가 소수점 표시가 안됨
     * **해결:** float을 사용하여 소수점이 나타날수 있도록 표현
  3. **문제 3:** 마지막 결과화면을 출력할때 수익률이 3자리 수인지 2자리 수인지에 따라 계속해서 결과값이 밀려 나타나는 현상 발생
     * **해결:** 수업시간에 배운 \t를 사용하거나 %10d를 사용하여 정리하려 했으나, 가로는 가독성이 떨어지는것 같아 결과값 화면을 세로로 수정
